### Design a browser extension that helps users stay focused by blocking access to selected social media websites during work sessions. The extension should include the following features

- A start/stop control that allows users to activate focus mode.
- When focus mode is active, specified websites should be inaccessible and display a custom message (e.g., 'Stay focused' or a simulated 'No internet' page).
- A settings panel where users can add or remove websites from the block list.
- A timer system based on the Pomodoro Technique (e.g., 25-minute work sessions followed by short breaks), with clear visual feedback on remaining time.
- A display showing how long the current session will block websites.
- Optional notifications or reminders when sessions start and end.

Provide a high-level architecture, key components, and recommended technologies for implementing this extension.
